# 1_termo
Material de aula para o primeiro termo - LOPAL - SOP - ARI - LER
# 📚 Grade Curricular: Engenharia, Infraestrutura e Desenvolvimento

Este repositório centraliza os conteúdos programáticos, códigos-fonte e materiais acadêmicos das quatro disciplinas fundamentais de tecnologia.

---

## 🛠️ 1. Levantamento de Requisitos
Focada nas técnicas de captação, análise e documentação de necessidades para o desenvolvimento de softwares alinhados ao negócio.
*   **Elicitação de Requisitos:** Aplicação prática de técnicas de *Brainstorming* (geração de ideias) e *Entrevistas* (estruturadas e abertas) com stakeholders.
*   **Classificação Técnica:** Divisão entre Requisitos Funcionais (RF - o que o sistema faz) e Não Funcionais (RNF - critérios de qualidade, desempenho e segurança).
*   **Validação e Modelagem:** Uso de *Prototipagem* (baixa e alta fidelidade via Figma) e diagramação técnica (Casos de Uso e Diagramas de Fluxo de Dados - DFD).
*   **Artefatos de Saída:** Elaboração de Relatórios Técnicos e da Especificação de Requisitos de Software (ERS).

## 🐍 2. Lógica de Programação com Python
Desenvolvimento de competências lógicas através do Python, integrando a criação de interfaces visuais a boas práticas de versionamento.
*   **Interfaces Gráficas (GUI):** Construção de telas modernas utilizando o ecossistema GNOME através do *PythonGi (PyGObject)*, manipulando loops de eventos, widgets e sinais.
*   **GitHub Clean:** Implementação de boas práticas de organização, uso estratégico de arquivos `.gitignore` para Python e padronização de histórico com *Conventional Commits*.
*   **Arquitetura de Projetos:** Transição de scripts simples para código modular, seguindo a estrutura padrão de diretórios (`src/`, `tests/`), semântica de variáveis e diretrizes da PEP 8.

## 🌐 3. Arquitetura de Redes IoT
Estudo da infraestrutura física e lógica necessária para conectar o mundo real à internet de forma automatizada e escalável.
*   **Infraestrutura de Rede:** Diferenciação e aplicação de componentes Ativos (roteadores, gateways IoT, switches) e Passivos (cabeamento estruturado e conectores).
*   **Evolução da Rede:** Análise da Internet Tradicional em contraste com a Internet das Coisas (IoT) e a Internet Industrial das Coisas (IIoT).
*   **Hardware e Drivers:** Pesquisa de dispositivos e parametrização do microcontrolador *ESP32* via drivers de comunicação serial-USB.
*   **Protocolo MQTT:** Arquitetura de comunicação leve baseada no modelo de Publicação/Assinatura (*Publish/Subscribe*) mediada por um Broker.

## 🖥️ 4. Sistemas Operacionais
Compreensão do funcionamento interno dos sistemas de computação, operação avançada via terminal e fundamentos de defesa cibernética.
*   **Ecossistemas e Distribuições:** Critérios de indicação entre os ambientes Windows (Client/Server) e as diversas distribuições Linux (Debian, Ubuntu, Kali, Red Hat).
*   **Operação via CLI (Prompt/PowerShell):** Gerenciamento avançado de diretórios, manipulação de variáveis de ambiente do sistema e controle de usuários/permissões via linha de comando.
*   **Segurança Cibernética:** Aplicação prática do Princípio do Menor Privilégio (PoLP), mecanismos de controle de acesso (UAC e NTFS) e técnicas de *Hardening* de S.O.
